# Zombie Top-Down Shooter

Мой первый проект в разработке. Простая 2D-стрелялка с видом сверху, где нужно выживать среди зомби.

## Игровые режимы:
1. **Выживание по времени**: Продержитесь от 5 до 10 минут против волн врагов.
2. **Зачистка**: Уничтожьте всех зомби на карте, чтобы победить.
3. **Бесконечный режим**: Классический тир. Зомби идут со всех сторон, игра идет на максимальный счет.

## О проекте
Это мой дебютный шаг в геймдеве. Код простой, проект создан ради изучения базовой логики перемещения, стрельбы и искусственного интеллекта врагов. Буду рад любой обратной связи!
